# Video_Playlist_Player
Video Playlist player built with Qt 6 and C++. This application provides dark themed interface with playback controls and Playlist Management.

Features

      Playback Controls
          Play/Pause - Smooth playback control
          Stop -  stop video playback
          Next/Previous - Navigate through your playlist effortlessly
          Shuffle Mode - Will Create Random playback  order.
          Prohress Bar - Go to any point in the video.
          AutoPlay Next - Automatically plays the next video when current ends.
          
     Audio
         Volume Control - Adjustable volume slider (0-100%)

     Playlist Management
        Add Multiple Videos - Select and add multiple video files at once
        Clear All - Clear entire playlist with one click
        Double-click to Play - Quick access to any video in the playlist
        Current Hihghlight - Current playing video is highlighted
        Remove Selected - Remove individual videos from playlist

    Supported Formats
        MP4 (.mp4)
        AVI (.avi)
        MKV (.mkv)
        MOV (.mov)
        WMV (.wmv)
        FLV (.flv)

Requirements

        Qt 6.x or higher
        C++ Compiler with C++17 support
                GCC 7+ (Linux)
                MSVC 2019+ (Windows)
       Qt Modules Required:
                Qt Core
                Qt Widgets
                Qt Multimedia
                Qt MultimediaWidgets
